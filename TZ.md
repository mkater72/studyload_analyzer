# Техническое задание на проект
## «StudyLoad Analyzer»

### 1. Цель проекта
Создание веб-сервиса для студентов, позволяющего анализировать учебную нагрузку и успеваемость за выбранный семестр. Система помогает визуализировать оценки, рассчитывать средний балл и выявлять наиболее нагруженные дисциплины.

### 2. Роли пользователей
- Гость: просмотр демонстрационных данных и аналитики.
- Администратор: управление предметами, семестрами и оценками через админ-панель Django.

### 3. Модели данных

**Semester (Семестр)**
- name — название семестра
- start_date — дата начала
- end_date — дата окончания

**Subject (Предмет)**
- name — название предмета
- weekly_hours — количество часов в неделю
- semester — связь с семестром (ForeignKey)

**Grade (Оценка)**
- value — значение оценки (2–5)
- date — дата получения
- subject — связь с предметом (ForeignKey)

### 4. Ключевой функционал
- Добавление и просмотр семестров и предметов
- Ввод оценок по каждому предмету
- Расчет среднего балла по предметам
- Расчет общей учебной нагрузки
- Визуализация данных в виде таблиц и графиков

### 5. Аналитика
Для анализа данных используется библиотека Pandas:
- расчет среднего балла
- агрегирование оценок
- подготовка данных для визуализации

Графики строятся с использованием Matplotlib.
